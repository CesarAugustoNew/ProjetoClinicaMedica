# 🏥 Projeto Clínica Médica

Sistema desenvolvido para gerenciamento de uma clínica médica.

## 📌 O que foi desenvolvido

Implementação de um sistema em Java utilizando conceitos de Programação Orientada a Objetos e CRUD, permitindo o cadastro, listagem, atualização e remoção de pacientes, médicos e consultas.
